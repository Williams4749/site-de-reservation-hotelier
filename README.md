# Site de Réservation Hôtelière

Application web de réservation hôtelière développée avec Django et MySQL.

## Technologies utilisées
- Python / Django
- MySQL
- HTML / CSS / JavaScript
- Django REST Framework
- Scikit-learn / Pandas / NumPy

## Fonctionnalités
- Recherche et réservation de chambres
- Gestion des disponibilités
- Interface utilisateur responsive
- consulter menus de restuarant
- commander un plat 

## Installation
1. Cloner le projet
2. Installer les dépendances : pip install -r requirements.txt
3. Configurer la base de données MySQL
4. Lancer : python manage.py runserver
